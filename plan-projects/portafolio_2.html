Plataforma de programación competitiva

Descripción general

Desarrollé una aplicación web enfocada en ejercicios de estructuras de datos, con énfasis en stacks y queues. El usuario puede elegir un ejercicio, revisar su enunciado, ver las pruebas unitarias esperadas y programar directamente en el navegador usando un editor de código integrado para Java. Al ejecutar, el sistema compila, corre los tests y reporta una calificación sobre 100 basada en los casos aprobados.

Listado de ejercicios

La pantalla principal presenta un catálogo de ejercicios categorizados por tema y dificultad, permitiendo seleccionar uno para profundizar en los requisitos y restricciones del problema.

Entorno de resolución y evaluación

Cada ejercicio incluye una descripción detallada, casos de ejemplo y pruebas unitarias públicas. El editor embebido permite escribir y ejecutar código Java sin salir de la página. Tras la ejecución, se muestran los resultados de las pruebas y la puntuación acumulada, lo que favorece una retroalimentación inmediata y un aprendizaje iterativo.

Características clave

El flujo integra edición, compilación y testeo automatizado. La arquitectura está pensada para facilitar la incorporación de nuevos ejercicios y la actualización de conjuntos de pruebas, con un énfasis en buenas prácticas de evaluación automática.
